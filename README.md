Email Sorter App

A Python-based application to automatically sort and manage emails using OAuth authentication. Users can log in via their email accounts securely without exposing passwords. 

---

Features

- Secure login with OAuth (no need to enter passwords in the app)
- Automatically sorts emails based on rules or categories
- Supports Gmail and other OAuth-compatible email providers
- Handles attachments and labels (if using Gmail)
- Logs activity and keeps track of sorted emails
